For Kinde to read and execute your workflow code, it needs to be structured a certain way. 

## Directory structure

```
myApp/
  ├── kindeSrc/
  │   └── environment/
  │       └── pages/
  │           └── (kinde)/
  │               └── <route>/
  │                   └── page.tsx
  ├── package.json
  └── kinde.json
```
